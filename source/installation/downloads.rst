GitHub
******

`Git <http://git-scm.com/about>`_ adalah sistem kontrol versi terdistribusi.

Akses *Public Git* tersedia di `GitHub <https://github.com/bcit-ci/CodeIgniter>`_.
Harap dicatat sementara bahwa setiap usaha dilakukan untuk menjaga *code base functional* ini, 
kami tidak dapat menjamin fungsi kode diambil dari *develop branch*.

Dimulai dengan versi 2.0.3, versi stabil juga tersedia melalui `GitHub Releases
<https://github.com/bcit-ci/CodeIgniter/releases>`_.
